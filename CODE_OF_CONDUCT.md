# 行動規範

## 私たちの約束

私たちはオープンで歓迎的な環境を作ることを約束します。すべての参加者が、経験やバックグラウンドに関わらず、ハラスメントのない体験を得られるようにします。

## 期待される行動

- 思いやりと敬意のある言葉を使う
- 異なる意見や経験を尊重する
- 建設的なフィードバックを歓迎する
- コミュニティにとって最善なことに焦点を当てる

## 許容されない行動

- 性的な言葉や画像の使用
- 侮辱的なコメントや個人攻撃
- ハラスメント（公的または私的）
- 他者の個人情報の無断公開

## 報告

問題のある行動を目撃した場合は、リポジトリの管理者に報告してください。

## 適用範囲

この行動規範は、プロジェクト空間内および、個人がプロジェクトやコミュニティを代表している公的空間に適用されます。
